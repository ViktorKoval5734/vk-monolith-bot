# Монолит - Бот для ВКонтакте

Мистический чат-бот с интеграцией YandexGPT для бесед ВКонтакте. Бот олицетворяет таинственную силу из вселенной S.T.A.L.K.E.R.

## Возможности

- Отвечает на упоминания в беседах
- Сохраняет историю переписки для контекста
- Использует API YandexGPT для генерации ответов
- Работает через Callback API (вебхуки)
- Случайные комментарии без упоминаний
- Резкие ответы на агрессивные сообщения
- Персонализация для конкретных пользователей

## Установка

1. Перейдите в директорию бота:
```bash
cd vk_monolith_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните токены:
```bash
cp .env.example .env
```

## Настройка ВКонтакте

1. Создайте сообщество в ВКонтакте (если ещё нет)
2. Получите токен сообщества:
   - Настройки → Работа с API → Long Poll API → создать токен
   - Разрешения: `messages`, `groups`
3. Включите Callback API:
   - Настройки → Работа с API → Callback API
   - Выберите версию API: `5.199`
   - **ВАЖНО:** Скопируйте код подтверждения, который показывает ВКонтакте
   - Скопируйте URL сервера (пока заглушка)

## Настройка YandexGPT

1. Перейдите в [Яндекс Облако](https://cloud.yandex.ru/)
2. Создайте аккаунт или войдите
3. Создайте сервисный аккаунт с правами `ai.languageModels.user`
4. Получите API ключ и ID папки
5. Добавьте их в `.env`:
   - `YANDEX_GPT_API_KEY` - ваш API ключ
   - `YANDEX_GPT_FOLDER_ID` - ID папки

## Запуск

### Локальный запуск (для тестирования)

1. Запустите туннель ngrok:
```bash
ngrok http 8000
```

2. Скопируйте HTTPS-URL от ngrok

3. В настройках Callback API ВКонтакте:
   - Адрес сервера: `https://ваш-домен.ngrok.io`
   - Секретный ключ: придумайте и сохраните в `.env`
   - **Код подтверждения:** скопируйте из настроек ВКонтакте

4. Обновите код подтверждения в `.env`:
```bash
VK_CONFIRMATION_CODE=код_из_ВКонтакте
```

5. Запустите бота:
```bash
python bot.py
```

6. Введите подтверждение в настройках Callback API

## Обновление кода подтверждения

Если код подтверждения изменился (например, при перенастройке туннеля):

1. **Автоматически:** Обновите переменную `VK_CONFIRMATION_CODE` в `.env` и перезапустите бота
2. **Через API:** Откройте `http://localhost:8000/update_confirmation/новый_код`
3. **Проверка:** Откройте `http://localhost:8000/confirmation_status` для проверки статуса

### Запуск на сервере

1. Настройте домен с HTTPS (Let's Encrypt)
2. Используйте nginx как reverse proxy
3. Настройте systemd сервис для автозапуска

## Команды бота

В беседе бот реагирует на упоминания:
- `@Монолит` или `[club|...]` - ответ на сообщение
- `Монолит, настройки` - показать список команд настройки
- `Монолит, имя <имя>` - установить особое обращение к вам

## Статистика и мониторинг

- `/deduplicator_status` - статистика предотвращения дублирования
- `/hostile_responses_status` - статистика резких ответов
- `/random_comments_status` - статистика случайных комментариев

## Характер бота

Монолит говорит загадочно, пророчески, использует термины Зоны:
- Аномалии, артефакты, выброс
- Сталкеры, фракции
- Сердце Зоны, исполнение желаний
