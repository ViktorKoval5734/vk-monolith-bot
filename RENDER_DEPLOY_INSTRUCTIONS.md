# 🚀 Инструкция по развертыванию бота "Монолит" на Render.com

Бот "Монолит" - это умный чат-бот для ВКонтакте с интеграцией YandexGPT. Он:
- Отвечает на упоминания в беседах ВКонтакте
- Использует AI (Гигачат) для генерации ответов
- Имеет систему персонализации для пользователей
- Поддерживает дедупликацию сообщений и системы защиты от спама

## 🛠️ Развертывание на Render.com

### Шаг 1: Создание сервиса

1. Перейдите на [render.com](https://render.com)
2. Войдите в аккаунт (GitHub, Google или email)
3. Нажмите **"New +"** → **"Web Service"**
4. Выберите **"Build and deploy from a Git repository"**

### Шаг 2: Подключение репозитория

1. Найдите и выберите ваш репозиторий с ботом
2. Нажмите **"Connect"**

### Шаг 3: Настройка сервиса

Заполните следующие поля:

**Basic Settings:**
- **Name**: `sota-sil-bot` (или любое другое уникальное имя)
- **Region**: Выберите регион ближе к вашим пользователям
- **Branch**: `main` (или `master`)
- **Root Directory**: оставьте пустым

**Runtime:**
- **Runtime**: `Python 3`
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `uvicorn bot:app --host 0.0.0.0 --port $PORT`

### Шаг 4: Настройка переменных окружения

В секции **"Environment"** добавьте следующие переменные:

**Обязательные переменные:**
```
VK_TOKEN=ваш_vk_token_здесь
VK_GROUP_ID=ваш_group_id_здесь
GIGACHAT_AUTH_KEY=ваш_auth_key_здесь
GIGACHAT_CLIENT_ID=ваш_client_id_здесь
VK_CONFIRMATION_CODE=код_подтверждения_из_вк
```

**Дополнительные переменные (опционально):**
```
GIGACHAT_SCOPE=GIGACHAT_API_PERS
CONFIRMATION_SECRET=ваш_секретный_ключ
HISTORY_FILE=history.json
PYTHON_VERSION=3.11.9
```

### Шаг 5: Настройка Auto-Deploy

- Включите **"Auto-Deploy"** для автоматических деплоев при коммитах

### Шаг 6: Создание сервиса

Нажмите **"Create Web Service"**

## ⚙️ Получение необходимых токенов

### 1. Токен ВКонтакте (VK_TOKEN)

1. Перейдите в ваше сообщество ВКонтакте
2. **Управление** → **Настройки** → **Работа с API**
3. **Long Poll API** → **Создать токен**
4. Выберите разрешения: `messages`, `groups`
5. Скопируйте токен

### 2. ID сообщества (VK_GROUP_ID)

1. В настройках сообщества найдите **"Адрес страницы"**
2. ID - это числовой идентификатор (без минуса)
3. Пример: если ссылка `vk.com/publicsotasil`, то ID = `123456789`

### 3. Токены Гигачата

1. Перейдите на [developers.sber.ru](https://developers.sber.ru)
2. **Каталог решений** → **GigaChat**
3. **Подключить** → **Создать приложение**
4. Получите:
   - **Authorization key** (для GIGACHAT_AUTH_KEY)
   - **Client ID** (для GIGACHAT_CLIENT_ID)
5. В настройках приложения установите **Scope**: `GIGACHAT_API_PERS`

### 4. Код подтверждения (VK_CONFIRMATION_CODE)

1. В настройках сообщества ВКонтакте:
2. **Работа с API** → **Callback API**
3. **Настройки сервера**
4. Временно укажите любой URL (например, `https://example.com`)
5. Скопируйте **"код подтверждения"**
6. Позже замените URL на реальный адрес Render

## 🔗 Настройка Callback API ВКонтакте

После успешного деплоя на Render:

### 1. Получите URL вашего сервиса

- В дашборде Render найдите ваш сервис
- Скопируйте **Public URL** (например: `https://sota-sil-bot.onrender.com`)

### 2. Настройте Callback API

1. В настройках сообщества ВКонтакте:
   - **Работа с API** → **Callback API** → **Настройки сервера**

2. Заполните:
   - **Адрес сервера**: `https://ВАШ_SERVE_URL.onrender.com`
   - **Секретный ключ**: совпадает с `CONFIRMATION_SECRET` в настройках
   - **Версия API**: `5.199`

3. **События**: выберите `message_new`

4. Нажмите **"Сохранить"**

### 3. Подтвердите сервер

1. Render автоматически отправит запрос подтверждения
2. В настройках Callback API нажмите **"Подтвердить"**
3. Если нужно, обновите код подтверждения в переменных окружения Render

## 🚨 Решение проблем

### Бот не отвечает

1. **Проверьте логи** в Render на наличие ошибок
2. **Убедитесь в правильности токенов** ВКонтакте и Гигачата
3. **Проверьте настройки Callback API** - правильный URL и события
4. **Проверьте код подтверждения** в переменных окружения

### Ошибки в логах

**"ModuleNotFoundError"**: Убедитесь, что все зависимости указаны в `requirements.txt`

**"Environment variable not set"**: Проверьте все обязательные переменные окружения

**"Invalid token"**: Проверьте правильность токенов ВКонтакте и Гигачата

### Callback API не работает

1. Убедитесь, что URL сервиса доступен (https://...)
2. Проверьте, что сервис запущен (зеленый статус в Render)
3. Попробуйте перенастроить Callback API заново
