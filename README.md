# Монолит - Бот для ВКонтакте

Мистический чат-бот с интеграцией YandexGPT для бесед ВКонтакте. Бот олицетворяет таинственную силу Монолита из вселенной S.T.A.L.K.E.R.

## Возможности

- Отвечает на упоминания в беседах
- Сохраняет историю переписки для контекста
- Использует API YandexGPT для генерации ответов
- Работает через Callback API (вебхуки)
- Случайные комментарии без упоминаний
- Резкие ответы на агрессивные сообщения
- Персонализация для конкретных пользователей

## Установка

1. Перейдите в директорию бота:
```bash
cd vk_monolith_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните токены:
```bash
cp .env.example .env
```

## Настройка ВКонтакте

1. Создайте сообщество в ВКонтакте
2. Получите токен сообщества: Настройки → Работа с API → Long Poll API → создать токен
3. Разрешения: `messages`, `groups`
4. Включите Callback API: Настройки → Работа с API → Callback API
5. Выберите версию API: `5.199`
6. Скопируйте код подтверждения из настроек ВКонтакте

## Настройка YandexGPT

1. Перейдите в [Яндекс Облако](https://cloud.yandex.ru/)
2. Создайте сервисный аккаунт с правами `ai.languageModels.user`
3. Получите API ключ и ID папки
4. Добавьте их в `.env`:
   - `YANDEX_GPT_API_KEY` - ваш API ключ
   - `YANDEX_GPT_FOLDER_ID` - ID папки

## Запуск

### Локальный запуск

1. Запустите туннель ngrok:
```bash
ngrok http 8000
```

2. Скопируйте HTTPS-URL от ngrok

3. В настройках Callback API ВКонтакте:
   - Адрес сервера: `https://ваш-домен.ngrok.io`
   - Секретный ключ: придумайте и сохраните в `.env`

4. Обновите код подтверждения в `.env`

5. Запустите бота:
```bash
python bot.py
```

## Команды бота

В беседе бот реагирует на упоминания:
- `@Монолит`, `Монолит`, `Wish Granter`, `Исполнитель Желаний`, `Сила`, `Эмиссар`, `Пророк`, `Голос Монолита`, `Великий Хранитель` или `[club|...]` - ответ на сообщение
- `Монолит, настройки` - показать список команд настройки
