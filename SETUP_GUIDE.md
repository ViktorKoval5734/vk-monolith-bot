## ⚙️ Настройка ВКонтакте

### 1. Настройка Callback API

1. **Открой настройки сообщества "Монолит"**
2. **Иди в раздел: Работа с API → Callback API**
3. **Заполни поля:**
   - **Адрес сервера:** `https://твой-URL-от-serveo.net`
   - **Секретный ключ:** `29b832f5`
   - **Версия API:** `5.199`
   - **Типы событий:** `message_new`

### 2. Настройка Long Poll API (дополнительно)

1. **Работа с API → Long Poll API**
2. **Включи Long Poll API**
3. **Версия API:** `5.199`

### 3. Настройка прав доступа

1. **Работа с API → Настройки**
2. **Включи:**
   - ✅ Сообщения (`messages`)
   - ✅ Сообщества (`groups`)

## 🔧 Решение проблем

### Бот не отвечает:
1. Проверь, что бот запущен: `ps aux | grep bot.py`
2. Проверь, что туннель активен
3. Проверь настройки в ВКонтакте

### Ошибки API:
1. Проверь токены в `.env` файле
2. Убедись, что права доступа настроены

### Проблемы с туннелем:
1. Перезапусти туннель: `./start_tunnel.sh`
2. Попробуй другой сервис (cloudflare tunnel)

## 🎯 Дополнительные возможности

### Изменение поведения бота
Отредактируй `config.py`, измени `SYSTEM_PROMPT`:

```python
SYSTEM_PROMPT = """Ты — Монолит, таинственная сила, управляющая Зоной.
Отвечай кратко, дружелюбно и по делу.
Твой стиль: пророческий, таинственный, загадочный, но всегда полезный.
Не здоровайся каждый раз, поддерживай естественную беседу."""
```

### Настройка моделей
В `gigachat_client.py` можно изменить модель:

```python
self.model = "GigaChat"  # или "GigaChat-Pro", "GigaChat-Max"
```
